# Wireless Mobile Phone Detector

## Overview
Wireless Mobile Phone Detector is an electronics project used to detect the presence of an active mobile phone nearby through radio frequency signals.

## Features
- Detects active mobile phones
- Simple electronic circuit
- Low power consumption
- Easy to use and portable

## Applications
- Examination halls
- Security systems
- Meeting rooms
- Restricted areas

## Technologies Used
- RF Signal Detection
- Electronics Components
- Circuit Design

## Conclusion
This project helps identify active mobile phones in nearby areas and can be used for security and monitoring purposes.
